# SI.MA Restyling

Progetto di restyling per SI.MA - sito web moderno e responsive.

## Struttura del Progetto

```
si.ma-restyling/
├── index.html          # Pagina principale
├── css/
│   └── style.css      # Stili CSS
├── js/
│   └── main.js        # JavaScript principale
├── .github/
│   └── copilot-instructions.md
└── README.md          # Questo file
```

## Caratteristiche

- Design moderno e pulito
- Completamente responsive
- Navigazione smooth scroll
- Layout a sezioni
- Ottimizzato per dispositivi mobili

## Come Aprire il Progetto

1. Apri il file `index.html` nel tuo browser
2. Oppure usa un server locale come Live Server in VS Code

## Tecnologie Utilizzate

- HTML5
- CSS3 (con CSS Variables e Flexbox)
- JavaScript (ES6+)

## Personalizzazione

- Modifica i colori in `css/style.css` (variabili CSS in `:root`)
- Aggiungi contenuti in `index.html`
- Estendi le funzionalità in `js/main.js`
